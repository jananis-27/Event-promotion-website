# Event Promotion Website 

A responsive and visually appealing event promotion website built using **HTML5** and **CSS3**.  
The project focuses on modern UI design, smooth animations, and clean layout practices suitable for beginners and internship-level portfolios.


## Features

- Modern and responsive UI design  
- Animated sparkle background using pure CSS  
- Floating images and hover animations  
- Shimmer text effect for headings  
- Card-based layout for better user experience  
- Hosted using GitHub Pages  

## Technologies Used

- HTML5  
- CSS3
